# GemOfLegend

Wrapper around the League of Legends (LoL) v1.2 api.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'GemOfLegend'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install GemOfLegend
